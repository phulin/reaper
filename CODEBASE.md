# Codebase Outline

This codebase uses progressive disclosure. Each package contains a more detailed README outlining each file's functionality and that of any subpackages.

* .agents/skills/: repo-local agent skills and bundled references
* scripts/: operational entrypoints and PyGhidra helpers for project creation, function recovery, inventories, bundles, string-to-code tracing, and applying reviewed naming artifacts
* src/reaper/: main package
